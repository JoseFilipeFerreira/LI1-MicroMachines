# LI1-MicroMachines

Haskell-Gloss MicroMachines (LI1 Assignment)

## Getting Started
### Setting up

* Make sure you have [Gloss](http://hackage.haskell.org/package/gloss) installed

```
ghc Tarefa5_2017li1g13
```

```
./Tarefa5_2017li1g13
```
